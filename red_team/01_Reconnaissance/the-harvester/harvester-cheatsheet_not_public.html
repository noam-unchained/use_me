The Harvester — cheat sheet

Passive OSINT tool for collecting emails and subdomains from Google, Bing, crt.sh, and HackerTarget.

Step 1 — Install & Run
pip install -r requirements.txt
python3 the_harvester.py -d example.com
Step 2 — Flags
FlagDescriptionExample
-dTarget domain (required)-d example.com
-sSources to use (default: all)-s google,crt
-oSave output to file-o results.txt
Step 3 — Sources
SourceWhat it collects
googleEmail addresses from Google search results
bingEmail addresses from Bing search results
crtSubdomains from certificate transparency logs (crt.sh)
hackertargetSubdomains from HackerTarget DNS lookup API
Step 4 & 5 — Reading Results
Emails found:
  admin@example.com
  info@example.com
  john.doe@example.com

Subdomains found:
  mail.example.com
  vpn.example.com
  dev.example.com

Emails feed into phishing simulation and credential guessing. Subdomains feed into further scanning.

Run Examples
# All sources
python3 the_harvester.py -d example.com

# Only cert transparency + bing
python3 the_harvester.py -d example.com -s crt,bing

# Save to file
python3 the_harvester.py -d example.com -o results.txt

Key idea: combine this with email-enum — use the harvester to find real email addresses and known subdomains, then use email-enum to guess additional addresses for people not yet found publicly.
