# Utilities

Supporting tools that are useful across multiple phases.

Nothing here is phase-specific — these are things you reach for throughout an engagement: encoding payloads, cracking captured hashes, auditing passwords, and cleaning up traces when you are done.

## Tools

| Tool | Purpose |
|------|---------|
| [encoder-decoder](encoder-decoder/) | Encode and decode data in base64, hex, URL, XOR, and more |
| [hash-cracker](hash-cracker/) | Crack hashes using wordlists and brute-force |
| [password-auditor](password-auditor/) | Audit password strength and check against common lists |
| [log-cleaner](log-cleaner/) | Clear system logs and shell history to remove traces |
| [phishing-detector](phishing-detector/) | Analyze a URL and score its phishing risk (0–100) |
| [linkedin_recon_not_public](linkedin_recon_not_public/) | OSINT enumeration of LinkedIn profiles and company employees |
