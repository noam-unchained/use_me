Password Auditor — cheat sheet

Analyzes a list of passwords from a dump or audit, grades each A–F, and generates a remediation report.

Step 1 — Run
# No pip install needed — pure Python 3
python3 password_auditor.py passwords.txt

# Show top N weakest
python3 password_auditor.py passwords.txt --worst 20

# Save report
python3 password_auditor.py passwords.txt -o report.txt
