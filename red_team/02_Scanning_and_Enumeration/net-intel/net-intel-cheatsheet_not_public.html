Net Intel — cheat sheet

Passive network intelligence tool. Analyzes pcap files or live traffic to extract credentials, map devices, and flag suspicious patterns.

Step 1 — Install Dependencies
# macOS
brew install wireshark
pip install -r requirements.txt

# Linux
sudo apt install tshark
pip install -r requirements.txt
Step 2 — Two Modes
# Analyze an existing pcap file
python3 net_intel.py pcap -f capture.pcap
python3 net_intel.py pcap -f capture.pcapng -o report.txt

# Live capture (requires root)
sudo python3 net_intel.py live -i eth0
sudo python3 net_intel.py live -i wlan0 -t 60 -o report.txt

-t sets capture duration in seconds. Omit for indefinite capture (Ctrl+C to stop).

Step 3 — Credential Extraction

Automatically extracts cleartext credentials from:

ProtocolWhat it extracts
HTTP Basic AuthUsername + password from Authorization header
FTPUSER and PASS commands
TelnetUsername and password from session
SMTP AUTHSMTP authentication credentials
HTTP POSTlogin/password fields from form submissions

Step 5 & 6 — DNS Tracking & Suspicious Flags

Every DNS query and response is logged. Suspicious patterns flagged automatically:

FlagWhat it means
DNS tunnelingUnusually large DNS queries — data may be exfiltrated via DNS
DGA domainsRandom-looking domain names — sign of malware C2
Cleartext cookiesSession cookies visible in HTTP traffic (no HTTPS)

Key idea: for live capture, use promiscuous mode on the interface for best coverage. Credentials only appear from unencrypted protocols — HTTPS traffic requires a MITM setup to inspect.
