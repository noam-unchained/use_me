Directory Enumeration — cheat sheet

Multithreaded web directory and file brute-forcer. Finds hidden paths, admin panels, and exposed files.

Step 1 — Install & Run
pip install -r requirements.txt

# Basic scan
python3 dir_enum.py -u http://target.com -w wordlist.txt

# With extensions
python3 dir_enum.py -u http://target.com -w wordlist.txt -e .php,.html,.bak

# Save results
python3 dir_enum.py -u http://target.com -w wordlist.txt -o found.txt

Step 4 — Reading Results

[301] http://target.com/uploads  ->  /uploads/
[403] http://target.com/.htaccess
[200] http://target.com/login.php      (4.8KB)
StatusMeaningAction
200Page exists and is accessibleInvestigate the content
301/302Redirect — note the destinationFollow the redirect
403Exists but access deniedTry other techniques to access
401Authentication requiredTry default creds

Key idea: start with a small wordlist (common.txt, ~4k entries) for a fast overview, then run a larger list (medium, ~220k) for thorough coverage. Add extensions relevant to the tech stack (-e .php for PHP apps).
